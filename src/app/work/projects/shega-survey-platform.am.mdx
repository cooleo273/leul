## እይታ

ሸጋ ሚዲያ እና ቴክኖሎጂስ የዳሰሳ መረጃ መሰብሰብ እና ሪፖርት ሂደትን ለማዋቀር ከ Google Forms ግልጽ የሆነ የሰነድ ሂደትን በሚቀይር ቀላል መድረክ ገነባሁ።

### ያደረግኩት

- በ Next.js እና React የተገነባ ምላሽ የሚሰጥ ፊት-መለኪያ ለዳሰሳ ፍጠር እና ስርጭት።
- Supabase በመጠቀም ደህንነቱ የተጠናቀቀ የመረጃ ማከማቻ እና እውነተኛ ጊዜ መከታተያዎች።
- ተግባራዊ ግምገማዎችን የሚያበረክቱ የሪፖርት ዳሽቦርዶች።

### ተፅዕኖ

ስርዓቱ የመረጃ ማካተትን እና ሪፖርት ሂደቶችን አውቶማቲክ ማድረግ ችሏል፣ የግብዓቶች ትንታኔ ጊዜን በአንድ 40% እንዳለ አሳነሰ።

## ቴክኖሎጂዎች

- Next.js, React
- Supabase
- PostgreSQL / Supabase
- Node.js

## ውጤት

የሥራ ሂደቶችን የሚቀላቀል እና ፈጣን የግብዓት ትንታኔ የሚያቀርብ ምርት-ዝግጁ መድረክ።

## ችግኝ

የሸጋ ቡድኖች ስለ ደንበኞች አስተያየት መረጃ በእጅ የሚከናወኑ ስራዎችን ይጠቀሙ ነበር፣ ይህም የሚስተው እና የጊዜ ማባከን ነበር።

## መፍትሔ

በፍጥነት ዳሰሳ ፍጠር፣ የታመነ ውሂብ መሰብሰብ፣ እና ተግባራዊ ሪፖርት ላይ ያተኮረ ቀላል መድረክ አበጀሁ። ሰራተኞች ዳሰሳዎችን ማቀናበር እና ማካፈል ይችላሉ፣ ባለሙያዎች ደግሞ በእውነተኛ ጊዜ የተθጣመ ሪፖርት ያያሉ።

## አወዳድሮች

```text
Browser (Next.js)  ->  API (Next API routes)  ->  Supabase (Postgres & Realtime)
                         |                             |
                         +--> Background workers / ETL --> Reporting DB / Materialized Views
```

### ዋና ክፍሎች
- ዳሰሳ መገንቢያ (drag-and-drop ቅደም ተከተል እና ቅድመ እይታ)
- ተሳታፊ ፊት-መለኪያ በቶከን መዳረሻ እና የእንግዳ መከላከያዎች
- እውነተኛ ጊዜ ማሰባሰብ እና ለሰርቨር ተለዋዋጭ ስራዎች
- ሪፖርት ዩአይ ከማጣቀሻዎች፣ ትርኢት ግራፎች እና CSV ላክ

## ባህሪዎች
- ብዙ የጥያቄ አይነቶች ያሉ ዳሰሳዎችን ፍጠር
- መርሐግብር እና የኢሜል ስርጭት መገናኛዎች
- በእውነተኛ ጊዜ የሚያዘመኑ ዳሽቦርዶች
- ውጤቶችን ወደ CSV እና ራስ-ሰር አጠቃላይ ኢሜሎች

## እይታ

ለ Shega Media & Technologies የፕሮጀክት ትርፍን እና የፋይናንስ አፈጻጸምን ለማስላት የውስጥ ዳሽቦርድ አቀናብረው አተገብርሁ።

### ያበጀኋቸው

- ወጪና ገቢን ወደ አንድ ምንጭ ለማካተት የዳታ ቧንቧዎች
- ለፕሮጀክት ደረጃ ትርፍ እና ትንበያ ቪዥዋላይዜሽን እና KPI
- ለኢኮኖሚስቶች እና ለአመራሮች ሪፖርቶች እና ወደ CSV ላክ

### ተፅዕኖ

አንድ ምንጭ ላይ መረጃን ማካተት የትንታኔ ትክክለኝነትን በ15%+ አሻሻለ እና ውሳኔን ፈጣን አደረገ።

## ቴክኖሎጂዎች

- Next.js, React
- PostgreSQL
- Node.js

## ውጤት

ለ ስራ ባለሙያዎች የፕሮጀክት ትርፍን እና አፈጻጸምን ለማስላት የሚረዳ የምርት-ዝግጁ መሳሪያ ተሰጠ።

## ችግኝ

ፋይናንስ እና የስራ ቡድኖች ትርፍን ለማስላት አንድ ምንጭ አልነበራቸውም። ስፕሬድሺቶች የተለያዩ ነበሩ እና ኮስት መመደብ የስህተት ምንጭ ነበር።

## መፍትሔ

የየቢል እና የጊዜ መዝገብ መረጃን በራስ-ሰር ማስገባት እና በቀላሉ የሚጠቀሙ ዩአይ ጋር ያቀረብ የውስጥ ዳሽቦርድ ገነብቻለሁ።

## አወዳድሮች

```text
Ingest (ETL) -> Central Postgres (normalized) -> API (Next.js) -> Frontend (Reports)
```

### ዋና ክፍሎች
- የ ETL ስራዎች እንደ የየቢል እና የጊዜ መዝገብ መረጃ
- የስር ሞዴል ለወጪ፣ ገቢ እና መመደቦች
- ትንበያ ሞዱል ለታሪካዊ ዝርዝር መሰረት ትንበያ
- በሚያስገባ መብት የተከፈለ ዩአይ እና የሪፖርት ላክ

## ባህሪዎች
- የውስጥ መለኪያዎች እና ትንሽ ኢኮኖሚክስ
- ምክንያታዊ ሁኔታዎች ለ 'what-if' ፕሮጀክሽኖች
- የተሰፋ ሪፖርቶች እና ኢሜል
